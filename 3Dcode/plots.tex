   Datasets
   \begin{itemize}
     \item all : all of the HCP dataset (12.0 fb$^{-1}$).
     \item RunsAB : 5.2 fb$^{-1}$.
     \item RunC : 6.8 fb$^{-1}$.
   \end{itemize}

.\\

   Other details
   \begin{itemize}
     \item MC has been corrected for trigger efficiency, {\it except} where noted otherwise.
     \item Trigger efficiency corrections are in likelihood MET,HT bins.
     \item Trigger efficiency corrections are not run dependent.
   \end{itemize}

.\\

  Some observations
  \begin{itemize}
    \item The best looking plots are the ones with no trigger corrections applied
          to the MC (i.e. trigger is 100\% efficient in MC), especially for the LDP.
    \item No big differences comparing RunsAB with RunC.
   \end{itemize}
